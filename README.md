## 本书主要内容
本书涉及到的知识主要包括：
PHP、Python、Nginx、Linux、MySQL、算法设计与分析、微服务、C++程序设计、Vue基础、系统架构、经典文摘、docker、设计模式等。

本节主要涉及机器环境搭建相关的一些问题，包括常用软件，服务等。
