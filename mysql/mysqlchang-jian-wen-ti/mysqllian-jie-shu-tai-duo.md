# **错误描述（MySQL无法连接\[MySql Host is blocked because of many connection errors\]）**

Host is blocked because of many connection errors; unblock with 'mysqladmin flush-hosts'

# **解决方案：**

方法一：

使用命令 \`mysqladmin flush-hosts\`
