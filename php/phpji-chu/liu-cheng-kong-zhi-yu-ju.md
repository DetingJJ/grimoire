流程控制语句是控制程序执行步骤的基本手段，也成为条件控制语句，主要有：
+ if语句
+ if...else语句
+ switch语句

+ if语句
if语句,也可以理解为单分支控制语句,语法格式:
```
<?php
    $a = true;     //声明布尔类型变量$a
    if($a){
        echo "条件为true";
    }
```
+ if...else语句
```<?php
$a = true;     //声明布尔类型变量$a
if($a){
    echo "条件为true";
}else{
    echo "条件为false";
}
```
